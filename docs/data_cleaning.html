Selección de países y periodo de análisis

Para mantener la visualización legible y comparativa, se seleccionaron cinco continentes y cinco países por continente. Además, se restringió el análisis al periodo 1990–2024 para trabajar con más de tres décadas de información y reducir problemas de cobertura incompleta en años más antiguos.

paises_por_continente = {
    "Africa": ["South Africa", "Egypt", "Algeria", "Morocco", "Nigeria"],
    "Asia": ["China", "India", "Japan", "South Korea", "Indonesia"],
    "Europe": ["Germany", "France", "Spain", "Norway", "Poland"],
    "North America": ["United States", "Canada", "Mexico", "Cuba", "Dominican Republic"],
    "South America": ["Brazil", "Argentina", "Chile", "Colombia", "Ecuador"]
}

paises = [pais for lista in paises_por_continente.values() for pais in lista]

df = df[
    (df["Entity"].isin(paises)) &
    (df["Year"] >= 1990) &
    (df["Year"] <= 2024)
].copy()

print("Dimensiones después del filtrado:", df.shape)
print("Países incluidos:", df["Entity"].nunique())
print("Años:", df["Year"].min(), "-", df["Year"].max())

Asignación de continente

Se creó una variable de continente para facilitar las comparaciones regionales.

mapa_continente = {}
for continente, lista in paises_por_continente.items():
    for pais in lista:
        mapa_continente[pais] = continente

df["Continent"] = df["Entity"].map(mapa_continente)

Manejo de valores faltantes

Los valores faltantes en las fuentes seleccionadas se reemplazaron por cero para mantener consistencia temporal entre países y años. Esta decisión se adoptó para evitar interrupciones en las series y facilitar la comparación visual.

df[fuentes] = df[fuentes].fillna(0)

Creación de variables auxiliares

Se añadieron variables útiles para el análisis comparativo, como la generación total por país y año, así como agregados por tipo de fuente.

df["Total generation"] = df[fuentes].sum(axis=1)
df["Fossil generation"] = df["Coal"] + df["Gas"] + df["Oil"]
df["Low carbon generation"] = df["Nuclear"] + df["Hydropower"] + df["Wind"] + df["Solar"]

Cálculo de participaciones porcentuales

Para algunas visualizaciones fue útil trabajar no solo con valores absolutos, sino también con la participación de cada fuente respecto al total de generación.

for fuente in fuentes:
    df[f"Share_{fuente}"] = df[fuente] / df["Total generation"].replace(0, pd.NA)

Transformación a formato largo para visualización

Como Altair trabaja especialmente bien con datos en formato largo, se transformó la tabla.

df_long = df.melt(
    id_vars=["Entity", "Continent", "Year", "Total generation", "Fossil generation", "Low carbon generation", "Dominant source"],
    value_vars=fuentes,
    var_name="Source",
    value_name="Generation TWh"
)

Justificación de las decisiones tomadas

Las decisiones de limpieza y transformación se tomaron con tres objetivos principales:

  • Asegurar comparabilidad entre países y continentes.
  • Mantener una cobertura temporal suficientemente amplia.
  • Construir visualizaciones claras para un público no técnico.

El análisis principal del proyecto se basa en una tabla limpia y estructurada para visualizaciones explicativas.
